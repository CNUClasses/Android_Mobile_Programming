4 4/25/20 Project Description A multithreaded networked application that uses ViewPager2 and a RecyclerView.Adapter. A port of project 3.
A base class to download an image using https If successful, the Image is returned as a BitMap. Subclass this class, and check to make sure that the Bitmap received is the same as what was originally requested by the Viewholder. Assummig it is (and the returned Bitmap is named result, and the ImageView is named iv) use the following call to set the Bitmap to the ImageView; iv.setImageBitmap(result)

Call notifyDataSetChanged() on the adapter you are using for ViewPager2 whenever the underlying data changes. When this project first starts you should display a single placeholder page. After which you should download JSON from your selected website. Once you parse this JSON you have a different number of pages to display. This is when you should call notifyDataSetChanged() on the adapter you are using for ViewPager2
