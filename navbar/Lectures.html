Builder Pattern Demo

Do not let users arbitrairly construct a complex object using variable length constructors, or getters and setters. These methods either do not scale or do not gurantee a properly constructed object. Instead make the objects constructor private and add a static class builder to it. The builder is responsible for collecting and verifing all fields of the object before the object is constructed. Once verified, the builder builds and returns an instance of the object through its build() method.
